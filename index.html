Background Image

To set a background image, you first need to find the image you want to use. Once you have found this, you need to set this image in the code. This is done by putting { backround-image: url("imageurl.png");}

Body

Where you put your background image depends on where and how you want it.

  • If you want it to be the background for the entire page, insert the { backround-image} inside of the body element.

    See the Pen by carson dudley (@cdudley6) on CodePen.

    P, div, header, etc..

  • If you want a background image just for a certain element, place it within your "p", "div", etc... element section in your code.

    See the Pen by carson dudley (@cdudley6) on CodePen.

Image Stacking

It is also possible to stack multiple images for your background. This is done by doing the same { backround-image: url("imageurl")code followed by a comma (the comma is to seperate the multiple backround images) and then the other url you want. You do need to make sure that this is done in the order that you want them stacked.

Example: { background-image: url("imageurl.png"), url("secondimageurle.png");
